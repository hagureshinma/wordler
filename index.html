Wordler

What is Wordler?

Wordler is a site to help people sutck finding a suitable word in Wordle.

When to use Wordler?

You should use this site only when you have tried at least three words and can't think of another one that fits the guessing criteria.

How to use Wordler?

Wordler it's simple to use, just write the letters you got right in Wordle in the same position here and press the button. A list of possible solutions should appear down, and you can click on each one to learn its meaning.

Why did you made this site?

I'm a Wordle enjoyer and did this proyect to learn JavaScript.

Thanks!

This site uses the dwyl / english-words repo to get the list of words. And the dictionaryapi.dev/ API to get word definitions.
